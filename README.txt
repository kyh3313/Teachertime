경북하이텍고 교사 시간표 PWA
==============================

배포 방법
---------
이 폴더의 모든 파일을 GitHub Pages 등 HTTPS 정적 웹 호스팅에 올리면 됩니다.

중요:
- index.html만 올리지 말고 폴더 전체를 같은 구조로 업로드하세요.
- iPhone의 홈 화면 설치와 오프라인 실행에는 HTTPS 주소가 필요합니다.
- 별도의 서버 프로그램이나 데이터베이스는 필요하지 않습니다.
- timetable.json 원본은 포함되어 있지 않습니다.
- 시간표는 encrypted-data.json에 AES-GCM 암호문으로만 저장되어 있습니다.

iPhone 설치 방법
----------------
1. Safari로 배포된 주소를 엽니다.
2. Safari 하단의 공유 버튼을 누릅니다.
3. '홈 화면에 추가'를 선택합니다.
4. 추가된 '교사 시간표' 아이콘으로 실행합니다.

기능
----
- 교사·학반 검색
- 한국 표준시 기준 현재 수업 표시
- 오늘/주간 시간표
- 전체 교사 현황
- HWPX 시간표 불러오기
- 불러온 시간표의 기기 내 저장
- 설치 후 오프라인 실행
- 비밀번호 잠금 및 15분 후 자동 잠금

주의
----
HWPX로 불러온 사용자 시간표는 해당 iPhone의 Safari/PWA 저장 공간에만 저장됩니다.
Safari의 웹사이트 데이터를 삭제하면 기본 시간표로 초기화됩니다.
비밀번호를 잊으면 암호화된 기본 시간표를 복구할 수 없습니다.
